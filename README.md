# OpenUrbanMap

OpenUrbanMap — это веб-приложение для визуализации и редактирования городских объектов и линий на карте.  
Проект ориентирован на работу с уличной инфраструктурой: дороги, линии, направления движения, элементы городской среды.

Проект разрабатывается как **open-source инструмент** с упором на:
- простую архитектуру
- наглядную визуализацию
- контроль над геометрией и стилями
- совместимость с OpenStreetMap-подходами
